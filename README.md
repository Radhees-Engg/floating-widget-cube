# Floating Cube 
This is a floating rotating cube that sits on your desktop and does absolutely nothing useful. and that's exactly the point.
it's transparent, borderless, hides from the taskbar, bobs up and down like it's vibing, and rotates forever. you can slap any texture on it. make it look like a minecraft block, a meme, your crush's face, whatever. no one can stop you.
